# BES Next.js SaaS Template

Production-ready Next.js SaaS starter template following the Brito Engineering Standard.

## Purpose

Use this template for SaaS products, internal tools, dashboards, portals, and business operating systems.

## Default Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- ESLint
- Prettier
- GitHub Actions
- Supabase-ready architecture
- Clean Architecture-inspired structure

## Source Structure

- src/app — Next.js App Router routes
- src/components — Reusable UI components
- src/features — Product features
- src/lib — Shared libraries
- src/services — External service integrations
- src/types — Shared TypeScript types
- src/config — App configuration
- src/utils — Utility functions

## BES Requirements

Every project generated from this template should include:

- Strict TypeScript
- Modular architecture
- No business logic inside UI components
- Tests for critical business logic
- Documentation for major systems
- CI checks before merge
- Release notes for meaningful changes
